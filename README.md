# Quickstart

### Run local mongo

    docker run -ti -p 27017:27107 mongo

### Run local node

    npm install -g standard nodemon
    npm install
    DB_IP=127.0.0.1 nodemon app.js

### Use todo.txt

  https://github.com/todotxt/todo.txt-cli

    brew install todo-txt
    alias t=todo.sh

    t help
